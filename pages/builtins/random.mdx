## random
returns a random item from given list
```py
random([1, 2, 3])
```

## randomNumber
returns a random number from given range
```py
randomNumber(1, 10)
```

## pyrandom
direct implementation of python random module
```py
pyrandom.choice([1, 2, 3])
pyrandom.randint(1, 10)
```
for more info on python random module, visit [python docs](https://docs.python.org/3/library/random.html)
