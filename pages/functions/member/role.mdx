## add_roles
add roles to a member

```python
add_roles(user, role_id_1, role_id_2, ...)
```

## remove_roles
remove roles from a member

```python
remove_roles(user, role_id_1, role_id_2, ...)
```

## has_role
check if a member has specific role

```python
has_role(user, role_id, role_id_2, ...)
```

## has_any_role
check if a member has any of the specific roles

```python
has_any_role(user, role_id_1, role_id_2, ...)
```
