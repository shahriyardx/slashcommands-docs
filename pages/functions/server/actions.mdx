# Channel actions
actions that can be done in channels

## create_role
creates a role
```python
create_role(role_name)
```

## delete_role
deletes a role
```python
delete_role(role_name | role_id)
```

## update_role
updates a role
```python
update_role(role_name | role_id, name="Changed name")
```

## create_text_channel
creates a text channel
```python
create_text_channel(channel_name)
```

## create_voice_channel
creates a voice channel
```python
create_voice_channel(channel_name)
```

## create_category
creates a category
```python
create_category(category_name)
```

## delete_channel
deletes a channel
```python
delete_channel(channel_name | channel_id)
```
